## Getting and Cleaning Data Course Project

This repo contains the analysis function *run_analysis.R* and the output of the analysis *tidydata.txt* from solving the course project for the Getting and Cleaning Data course from the Data Specialization Trach on Coursera.

The instructions for the Project can be found here:
https://class.coursera.org/getdata-007/human_grading/view/courses/972585/assessments/3/submissions

The repo also contains a CodeBook.md, a code book that describes the variables, the data, and any transformations or work that you performed to clean up the data called CodeBook.md

All the data for running the script is contained inside the folder *UCIHARDDataset/*
